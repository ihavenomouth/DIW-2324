Partiendo del truco para mostrar contenido oculto de manera suave, crea un menú con submenú.

HTML:

<div class="wrapper">
  <div class="inner">Expandable content</div>
</div>

CSS:

.wrapper {
  display: grid;
  grid-template-rows: 0fr;
  transition: grid-template-rows 0.5s ease-out;
}

.wrapper.is-open {
  grid-template-rows: 1fr;
}

.inner {
  overflow: hidden;
}
