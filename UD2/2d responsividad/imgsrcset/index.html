Imágenes responsivas

Se usa el atributo srcset para indicar resoluciones diferentes de la misma imagen. Evidentemente se puede cambiar también la imagen, para crear imágenes responsivas, pero sólo las podrá seleccionar correctamente el navegador si tienen la misma relación de aspecto (la misma relación entre el ancho y el alto de la imagen).

En el ejemplo de abajo la imagen elegida ocupa un 50% del ancho del viewport por defecto. Cuando éste tenga un tamaño menor que 650px, se usará la imagen de menor resolución que ocupará el 30% del ancho del viewport.

Se usa la etiqueta picture para indicar imágenes diferentes a utilizar en función del ancho de la pantalla.

Esto puede ser útil para mostrar un logotipo grande si la web está siendo accedida desde el escritorio, pero uno pequeño o un isotipo si está siendo accedida desde un móvil.

Además también contribuye a reducir el ancho de banda al cargar una imagen más pequeña.
