Diseño web al alcance de todos

Hacer un diseño de una web nunca fue tan sencillo.

Bootstrap simplifica la creación y organización de los elementos en la página sin ocultar el HTML ni el CSS.

Si necesitas hacer uso de CSS personalizado a algún elemento, puedes crear tu propio fichero de hoja de estilos.
Totalmente responsive

La organización de los elementos en la página se adapta al tamaño de la pantalla del dispositivo.

Con Bootstrap podemos crear una web o una aplicación que se adapte al dispositivo que esté usando el usuario.

Por ejemplo, la imagen que acompaña a este texto desaparece si la ventana es lo suficientemente pequeña.
